## 🔧 核心算法实现

### 1. 关键路径算法（CPM）
```
- 拓扑排序（Kahn算法）
- 最早开始时间（ES）计算
- 最晚开始时间（LS）计算
- Slack时间 = LS - ES
- 关键路径 = Slack为0的任务链
```

### 2. 循环依赖检测（DFS）
```
- 深度优先搜索
- 访问标记
- 环检测
```

### 3. 优先级分配算法
```
- 智能分配任务优先级
- 基于位置的优先级分布
- 支持拖拽重排序
```

