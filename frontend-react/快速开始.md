# 🚀 快速开始指南

## 前置要求

1. **Node.js** - 建议使用 Node.js 18+（当前使用的是 v16，可能会有警告，但可以运行）
2. **后端服务** - Spring Boot 应用运行在 `http://localhost:8080`

## 步骤 1：安装依赖

```bash
cd frontend-react
npm install
```

## 步骤 2：启动后端服务（如果还没启动）

```bash
# 在另一个终端窗口
cd ../spring-boot-demo
mvn spring-boot:run
```

等待看到：
```
Started SpringBootDemoApplication in X.XXX seconds
```

## 步骤 3：启动前端开发服务器

```bash
# 在 frontend-react 目录下
npm run dev
```

前端将在 `http://localhost:5173` 运行（Vite 默认端口）

## 步骤 4：访问应用

打开浏览器访问：`http://localhost:5173`

## 📋 使用流程

### 1. 注册新用户
- 点击"注册"标签
- 填写用户名、年龄、邮箱、密码
- 点击"注册"按钮

### 2. 登录
- 使用注册的账号登录
- 登录成功后自动跳转到首页

### 3. 查看用户列表
- 点击左侧菜单"用户管理"
- 查看所有用户列表
- 可以搜索、创建、编辑、删除用户

### 4. 修改密码
- 点击右上角用户头像
- 选择"修改密码"
- 输入原密码和新密码

## 🐛 常见问题

### 1. npm install 失败
- 检查网络连接
- 尝试使用国内镜像：`npm install --registry=https://registry.npmmirror.com`

### 2. 无法连接后端
- 检查后端是否运行：访问 `http://localhost:8080/hello`
- 检查浏览器控制台（F12）是否有错误
- 确认 API 配置：`src/config/api.js`

### 3. CORS 错误
- 后端已经配置了 CORS，应该不会有问题
- 如果还有问题，检查 `WebConfig.java` 中的 CORS 配置

### 4. 页面空白
- 检查浏览器控制台错误
- 确认所有依赖都已安装
- 尝试清除浏览器缓存

### 5. Node 版本警告
- 当前使用的是 Node v16，新版本的 Vite 需要 Node 18+
- 警告不影响运行，但建议升级 Node.js
